# Smart Snippets Library

A collection of C++ snippets for developers to write code faster and reduce errors.

## Snippets

### smart_placeholder.cpp
Automatically generates placeholder values for int, double, string types for testing and prototyping.

### auto_debug_hint.cpp
Automatically prints debug hints for floating-point differences.

### double_epsilon_check.cpp
Avoid floating-point equality issues with double values using epsilon.
